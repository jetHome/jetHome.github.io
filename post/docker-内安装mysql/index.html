docker 内安装mysql

一 启动mysql容器

1.拉取mysql镜像

2.根据镜像启动容器

docker run --name myMysql \

-p 33306:3306 \

-v /my/mysql/data:/var/lib/mysql \

-e MYSQL_ROOT_PASSWORD=root \

-d
mysql:my

3.确认容器启动

docker p

二 调整mysql的配置

1.进入mysql容器

docker ps -a
docker exec -it 容器id /bin/bash

2.如果容器内没有vim，执行如下操作

apt-get update
apt-get install vim

3.修改msysql配置文件

4.重新启动容器

docker ps
docker stop 容器id
docker start 容器id
